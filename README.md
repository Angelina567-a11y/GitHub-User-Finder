# GitHub User Finder

Автор: Ваше Имя и Фамилия

## Описание
Это GUI-приложение для поиска пользователей GitHub через API, просмотра результатов и добавления избранных пользователей. Избранные сохраняются в JSON.

## Использование API
- Введите имя пользователя в поле поиска.
- Нажмите кнопку "Поиск".
- Посмотрите результат и добавьте в избранное, нажав "Добавить в избранное".
- Для просмотра избранных нажмите "Показать избранных".
- Можно удалять пользователей из списка избранных.

## Запуск
Просто запустите основной скрипт и следуйте подсказкам.

## Замечания
- API GitHub имеет лимит 60 запросов в час для неавторизованных запросов.
- Для больших проектов рекомендуется использовать аутентификацию.
